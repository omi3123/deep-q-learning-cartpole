# Deep Q-Learning on CartPole

A portfolio-ready reinforcement learning project that demonstrates how a Deep Q-Network (DQN) can learn to solve the CartPole environment.

This project focuses not only on training the agent, but also on presenting the learning process clearly through visualizations, evaluation summaries, and concise interpretation of results.

## Project Overview

CartPole is one of the most common benchmark problems in reinforcement learning. The goal is to keep a pole balanced on a moving cart for as long as possible by choosing left or right actions at each step.

In this project, a DQN agent is trained on the CartPole environment and then evaluated using performance metrics and visual analysis.

## Project Objectives

- Understand how Deep Q-Learning works in practice
- Train a DQN agent on the CartPole environment
- Track and visualize training progression
- Compare training and evaluation performance
- Present the project in a clean portfolio-ready format

## Features

- Deep Q-Learning implementation on CartPole
- Reward progression analysis
- Episode length tracking
- Reward distribution visualization
- Training phase comparison
- Evaluation score summary
- Training vs evaluation comparison
- Hyperparameter summary
- Final project insights and interpretation

## Workflow

1. Set up the CartPole environment
2. Initialize the DQN model
3. Train the agent over multiple timesteps
4. Track reward and performance trends
5. Evaluate the trained policy
6. Visualize results with professional plots
7. Summarize findings and interpret agent behavior
